# Attendance Automation Bot

A Python-based automation tool that automatically performs employee attendance actions such as Clock In and Clock Out using Selenium WebDriver.

The bot logs into a web-based attendance portal, performs attendance actions, and can optionally send email notifications upon successful execution.

## Features

* Automated login to attendance portal
* Clock In and Clock Out functionality
* Secure credential management using environment variables
* Email notifications for successful attendance actions
* ChromeDriver auto-management using webdriver-manager
* Windows Task Scheduler integration for fully automated daily execution

## Tech Stack

* Python
* Selenium
* ChromeDriver
* dotenv
* yagmail

## Use Cases

This project demonstrates browser automation, workflow automation, scheduled task execution, and secure configuration management.

## Disclaimer

This project is intended for educational and automation-learning purposes. Ensure compliance with your organization's policies before automating attendance systems.
